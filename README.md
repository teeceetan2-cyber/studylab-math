# 📐 StudyLab — Math

Interactive mathematics learning app built with Streamlit.

**Topics:**
- Quadratic Equations — solve ax²+bx+c with graphing
- Trigonometry Explorer — sin, cos, tan with amplitude/frequency/phase
- Derivative Visualizer — compare functions and their derivatives
- Integral Area — visualize and compute definite integrals
- Unit Circle — explore sine, cosine, tangent interactively
- 3D Geometry — Sphere, Cube, Pyramid, Prism, Cylinder, Cone, Torus, Helix, Sinusoidal Surface

### Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

Or deploy on **Streamlit Cloud**: connect this repo and set entry point to `app.py`.
